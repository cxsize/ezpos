# Fonts

The app uses these TTFs — drop them in this folder (filenames must match):

- `Geist-Regular.ttf` — [Geist](https://github.com/vercel/geist-font)
- `Geist-Medium.ttf`
- `BodoniModa-Regular.ttf` — [Bodoni Moda](https://fonts.google.com/specimen/Bodoni+Moda)
- `NotoSansThai-Regular.ttf` — [Noto Sans Thai](https://fonts.google.com/noto/specimen/Noto+Sans+Thai)
- `NotoSerifThai-Regular.ttf` — [Noto Serif Thai](https://fonts.google.com/noto/specimen/Noto+Serif+Thai)

All are OFL-licensed. They're referenced from `app/_layout.tsx` via
`expo-font`. Missing files will fail the first launch loudly.
